// 时间序列数据，按时间桶聚合

?metric=ttft | tps | tokens | cache_rate | request_count

图表库选择

A

Chart.js (vue-chartjs)

轻量（~65KB gzip），生态成熟，适合折线/柱状图。社区最大，文档丰富

B

Apache ECharts (vue-echarts)

功能更强（~300KB gzip），交互丰富，内置 tooltip/缩放/数据筛选。但体积大
